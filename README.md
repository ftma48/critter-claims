# Critter Claims
Cute homes. Costly mistakes.

## Overview
Critter Claims is a point-and-click educational game created for HackSussex Hackathon 2026, under Hastings Direct’s challenge: “Make insurance claims fun and less stressful.”

Players choose from three animal characters and explore their homes, encountering household disasters and learning when and how to make insurance claims.

## Controls
- Mouse: Navigate UI and interact
- Enter: Progress dialogue

## Built With
- Godot (version 4.4)
- GDScript

## How to run
1. Download Godot 4
2. Clone this repository.
3. Open the project in Godot.
4. Press F5 to run.
